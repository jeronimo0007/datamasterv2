Como iniciar

1
Abra o Docker Desktop

Aguarde o daemon ficar pronto (docker ps sem erro).

2
Fluxo completo (sempre)

JSON → MongoDB user_profiles → lake Bronze/Silver/Gold:

cd datamaster
bash scripts/run_demo.sh

Ou use o botão abaixo no portal (mesmo fluxo via console :3333).

3
Só subir containers (sem dados) 
docker compose up -d --build
4
Postgres + Grafana (se volume antigo) 
bash scripts/seed_postgres.sh
docker compose up -d grafana

Postgres: transactions, fraud_alerts · Grafana: pasta DataMaster.

5
Explore pelos links abaixo

Ordem na banca: fluxo completo → API /analyze → Dashboard → Grafana.
